```javascript
あ a 类似“安”，常见单词“ありがとう：谢谢”
い i 类似“以”，常见单词“はい：是的”
う u 类似“乌”的上半段
え e 类似“诶”，有两个脚
お o 类似“拗”，看起来很绕很拗
```

```
か ka 类似“咖”
き ki 字形跟さ(sa)很像，像一个钥匙(key)
く ku 像一个人被从腰间踹了一脚，然后开始哭(ku)
け ke 类似“开”
こ ko 类似“口”，常见单词“こにちは：你好”
```

```
さ sa 类似“杀”，上面都有个×，常见单词“さようなら：再见”
し shi 像吸管，常见单词“よし：好”
す su 像丝瓜藤，常见单词“すみません：对不起”
せ se 类似“世”，常见单词“せんせい：老师”
そ so 像一个前凸后翘的人，so
```

```
た ta 类似“达”
ち chi 像数字5，但是是7，常见单词“こにちは：你好”
つ tsu (ci)出来一个圆
て te 类似“台”，常见单词“やめて：不要啊”
と to 像用手托着一个塔
```

```
な na 类似“奈”，常见单词“さようなら：再见”
に ni 类似“尼”，常见单词“こにちは：你好”
ぬ nu 类似“奴”
ね ne 类似“奶”，字形跟わ(wa)很像，区别在于最后一笔是否绕个圈
の no 类似“No”
```

```
は ha 字形跟ほ(ho)很像：は 哈！ほ 吼！常见单词“はい：是的”，有些时候读wa，常见单词“こにちは：你好”
ひ hi 像一个“嘿嘿”笑起来的嘴
ふ fu 有点像“夫”，小丈夫，读音也介于hu和fu之间
へ he 像一个人在鞠躬，说“はい：是的”
ほ ho 类似“耗”
```

```
ま ma 类似“麦”，常见单词“すみません：对不起”
み mi 类似“美”，写出来就是21，常见单词“すみません：对不起”
む mu 字形跟お(o)很像，区别在于最后一笔向上翘还是向下弯，类似“姆”写了一半没写完
め me 字形跟あ(a)很像，常见单词“やめて：不要啊”
も mo 类似“毛”
```

```
や ya 类似“也”，常见单词“やめて：不要啊”
ゆ yu 类似“由”
よ yo 像腰很细的人，所以读yo，常见单词“よし：好”，“さようなら：再见”
```

```
ら ra 字形跟ち(chi)很像，有点像数字6，6对应音阶里的la，常见单词“さようなら：再见”
り ri 类似“力”，li刀旁
る ru 下面像伸着舌头，发出“噜噜噜”的声音
れ re 类似“来”，右边像一个手在招揽，做出招呼人来的手势
ろ ro 类似る(lu)，漏了个舌头，读lo
```

```
わ wa 字形跟れ(re)很像，区别在于最后一笔向上翘还是向下弯，右边像一只手在挖东西
を wo 像一个人站在车上大喊“喔喔喔”，但是这个读音跟お一样是o，不是wo
```

```
ん nn 类似“n”，常见单词“せんせい：老师”
```
